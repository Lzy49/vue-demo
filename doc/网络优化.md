# 网络缓存
## 强缓存和协商缓存
- 发一个请求 -> no-store 是否存在
  - 是: [无缓存]
  - 否: no-cache 是否存在->
    - 是: [协商缓存]
    - 否: 验证是否是强缓存
      - cache-control 是否在这个时间范围内 -> expires 是否在该日期前:
        - 是: [强缓存]
        - 否: [协商缓存]
- [无缓存] -> 跨过缓存服务器 -> 直拿第一线
- [强缓存] -> 读取本地文件
- [协商缓存] -> 浏览器携带Last-Modified 和 Etag 讯问服务器是否更新 -> 服务器对比文件 Last-Modified 和 Etag
  - 更新 -> 200 -> 文件 + Last-Modified 和 Etag
  - 不更新 -> 304 -> 取本地文件
